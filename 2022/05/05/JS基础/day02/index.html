运算符

算术运算符

数学运算符也叫算术运算符，主要包括加、减、乘、除、取余（求模）。

false 参与算数运算被当做 0

执行优先级顺序

同时使用多个运算符编写程序时，会按着某种顺序先后执行，我们称为优先级。

JavaScript 中 优先级越高越先被执行，优先级相同时以书写顺序从左向右执行。

赋值运算符

赋值运算符：对变量进行赋值的运算符

一元运算符

自增：

l 符号：++

l 作用：让变量的值 +1

自减：

符号：- -

作用：让变量的值 -1

使用场景：经常用于计数来使用。 比如进行 10 次操作，用它来计算进行了多少次了。

前置，后置自增

前置自增和后置自增单独使用没有区别

前置自增和后置自增如果参与运算就有区别

注意：

l 前置自增和后置自增独立使用时二者并没有差别！

l 一般开发中我们都是独立使用

l 后面 i++ 后置自增会使用相对较多

比较运算符

比较运算符的使用：

比较运算符的细节：

l 字符串比较，是比较的字符对应的 ASCII 码

n 从左往右依次比较

n 如果第一位一样再比较第二位，以此类推

n 比较的少，了解即可

l NaN 不等于任何值，包括它本身

l 尽量不要比较小数，因为小数有精度问题

l 不同类型之间比较会发生隐式转换

n 最终把数据隐式转换转成 number 类型再比较

n 所以开发中，如果进行准确的比较我们更喜欢 === 或者 !==

逻辑运算符

逻辑运算符的使用：

注意：有 5 个值是当 false 来看的，其余都是真的。

false , 数字 0 , ‘ ‘ , undefined , null , NaN

逻辑运算符里的短路：

l 短路：只存在于 && 和 || 中，当满足一定条件会让右边代码不执行, 直接反馈短路的值。

l 不短路时以后面的值为准。

l 原因：通过左边能得到整个式子的结果，因此没必要再判断右边，就算有两个&&或者 || 只要前面的发生短路中断了，后面就不用判断了。

l 运算结果：无论 && 还是 || ，运算结果都是最后被执行的表达式值，一般用在变量赋值

l 左边为变量赋的值，右边为给的默认值。

额外内容：判断一个数是否整除案例

当用户不输入值时：

运算符优先级

l 一元运算符里面的逻辑非优先级很高

l 逻辑与比逻辑或优先级高

练习：

let a = 3 > 5 && 2 < 7 && 3 == 4 // 答案是 false，此时发生了逻辑与中断

console.log(a);

let b = 3 <= 4 || 3 > 1 || 3 != 2 // 答案是 true，此时发生了逻辑或中断

console.log(b);

let c = 2 === “2” // 答案是 false，数据类型不同

console.log(c);

​

let d = !c || b && a // 答案是 true，此事发生了逻辑或中断

console.log(d);

分支语句

程序三大流程控制语句

l 以前我们写的代码，写几句就从上往下执行几句，这种叫顺序结构

l 有的时候要根据条件选择执行代码，这种就叫分支结构

l 某段代码被重复执行，就叫循环结构

n If 分支语句

n switch 语句

if 分支语句

l if 语句有三种使用：单分支、双分支、多分支

l js 里规定，如果后面只有一个语句可以省略花括号，但是推荐都写花括号。

n 括号内的条件为 true 时，进入大括号里执行代码

n 小括号内的结果若不是布尔类型时，会发生隐式转换转为布尔类型

l 多分支 if 语法：

三元运算符

l 其实是比 if 双分支 更简单的写法，有时候也叫做三元表达式

l 符号：? 与 : 配合使用

l 语法：

l 一般用来取值：可以直接赋值给一个变量

n 案例：判断输入的两个数的最大值

switch 语句

释义：

l 找到跟小括号里数据全等的 case 值，并执行里面对应的代码

l 若没有全等 === 的则执行 default 里的代码，default 语句不是必须要写的，可以不写

l 例：数据若跟值 2 全等，则执行代码 2

注意事项：

l switch case 语句一般用于等值判断,不适合于区间判断

l switch case 一般需要配合 break 关键字使用 没有 break 会造成 case 穿透（一直往下执行，直到碰到 break）

l 可应用于分支较多的范围取值判定

循环语句

断点调试

l 作用：学习时可以帮助更好的理解代码运行，工作时可以更快找到 bug

l 浏览器打开调试界面

n 按 F12 打开开发者工具

n 点到 sources 一栏

n 选择代码文件

l 断点：在某句代码上加的标记就叫断点，当程序执行到这句有标记的代码时会暂停下来

while 循环语法：

释义：

l 跟 if 语句很像，都要满足小括号里的条件为 true 才会进入执行代码

l while 大括号里代码执行完毕后不会跳出，而是继续回到小括号里判断条件是否满足，若满足又执行大括号里的代码，然后再回到小括号判断条件，直到括号内条件不满足，即跳出

while 循环注意事项：

循环的本质就是以某个变量为起始值，然后不断产生变化量，慢慢靠近终止条件的过程。

所以，循环需要具备三要素：

\1. 变量起始值

\2. 终止条件（没有终止条件，循环会一直执行，造成死循环）

\3. 变量变化量（用自增或者自减

循环退出

l 循环结束：

n continue：结束本次循环，继续下次循环

u 在 i 加到 3 时 if 条件成立，使得 i++，并且结束从这次循环直接进行下次循环，由此跳过第三次的吃包子。

n break：跳出所在的循环，即退出循环。

do-while 循环

do… while 语句其实是 while 语句的一个变体。该循环会先执行一次代码块，然后对条件表达式进行判断，如果条件为真，就会重复执行循环体，否则退出循环。

do… while 语句的语法结构如下：

do {

// 循环体代码 - 条件表达式为 true 时重复执行循环体代码

} while(条件表达式);

do-while 循环 案列

// while 循环案例

// 1. 打印人的一生，从 1 岁到 100 岁

let i = 1;

do {

console.log(‘这个人今年’ + i + ‘岁了’);

i++;

} while (i <= 100)

// 2. 计算 1 ~ 100 之间所有整数的和

let sum = 0;

let j = 1;

do {

sum += j;

j++;

} while (j <= 100)

console.log(sum);

// 3. 弹出一个提示框， 你爱我吗？ 如果输入我爱你，就提示结束，否则，一直询问。

do {

let message = prompt(‘你爱我吗?’);

} while (message !== ‘我爱你’)

alert(‘我也爱你啊’);
